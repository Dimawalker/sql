Описание проекта

Проект представляет собой комплексную базу данных для управления информацией о транспортных средствах (автомобили, мотоциклы, велосипеды), а также для отслеживания результатов автомобильных гонок. Система позволяет анализировать производительность различных классов автомобилей в гонках и предоставляет инструменты для сравнительного анализа.

Цели и задачи проекта

Цели

Создание универсальной базы данных для различных типов транспортных средств

Разработка системы отслеживания результатов гонок

Реализация аналитических запросов для оценки производительности

Задачи

1. проектирование структуры для хранения информации о разных типах транспортных средств

2. создание системы классов и гонок

3. Реализация сложных аналитических запросов для:

4. Поиска оптимальных автомобилей по критериям

5. Анализа средней позиции в гонках

6. Сравнения производительности классов

7. Выявления лидеров в различных категориях
